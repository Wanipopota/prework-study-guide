HTML

  • HTML is the standard markup language used for creating web pages and web applications.
  • HTML documents are composed of elements, which are defined by tags. Tags are enclosed in angle brackets (< >) and typically come in pairs: opening tags and closing tags.
  • The head section typically contains metadata such as the document's title, links to external stylesheets or scripts, and other essential information for the browser.
  • The body section contains the content of the web page, including text, images, multimedia, and other elements.
  • HTML introduced semantic elements like header, nav, main, section, article, footer, etc., which provide meaning and structure to the content.
  • HTML supports ordered lists (ol), unordered lists (ul), and definition lists (dl) for organizing and presenting information in a structured manner.

CSS

  • CSS is a style sheet language used for describing the presentation of a document written in HTML. It defines how elements should be displayed on a web page.
  • CSS properties define the visual appearance of HTML elements. They include attributes such as color, font size, margin, padding, border, background, and more.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS properties are assigned values that determine their specific appearance. Values can be keywords, numerical values (e.g., pixels, percentages), colors, or other units of measurement.
  • A set of CSS properties and their values enclosed within curly braces ({ }) is called a declaration block. Multiple declaration blocks can be applied to a single selector.

Git

  • Git is a distributed version control system used for tracking changes in files and coordinating work among multiple developers.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: command to commit all files in the working branch.
  • git commit -m “<>”: to associate a message with our commit.
  • git pull origin main: need to sync our local branch with the remote branch in GitHub.
  • git push origin feature/BRANCH NAME: push up all of our changes.

JavaScript

  • JavaScript is a high-level, interpreted programming language primarily used for adding interactivity and dynamic behavior to web pages.
  • Variables are containers for storing data values. In JavaScript, variables can be declared using keywords like "var", "let", or "const", and can hold various types of data, including numbers, strings, booleans, arrays, objects, and functions.
  • JavaScript supports various data types, including primitive types such as numbers, strings, booleans, null, undefined, as well as complex types like objects, arrays, and functions.
  • Operators are symbols used to perform operations on variables and values. JavaScript supports arithmetic operators (+, -, *, /), assignment operators (=), comparison operators (==, ===, !=, !==), logical operators (&&, ||, !), and more.
  • Control flow is the order in which a computer executes code in a script.
  • Functions are reusable blocks of code that perform a specific task.
  • Math.floor() and Math.random() are both properties of JavaScript's built-in Math object that can be used together to generate a random number.
